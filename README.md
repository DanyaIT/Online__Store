## Online__Store Онлайн магазин для розничной торговли. Позволяет добавлять, просмативать, искать, оценивать товар.
# Cтек: Node.js,React, Express, Postgresql, Sequelize.
# Старт
  1. Frontend: npm start
  2. Backrend: npm run dev
